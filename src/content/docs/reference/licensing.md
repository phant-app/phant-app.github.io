Phant is source-available software licensed under the Business Source License 1.1.

## What this means

This page is a short, plain-language summary only.

The canonical license text is in the main Phant repository:

- https://github.com/phant-app/phant/blob/main/LICENSE.md

## Current license summary

Phant uses the Business Source License 1.1 with an Additional Use Grant.

According to the current license text, the Additional Use Grant allows use of the licensed work for:

- non-commercial purposes
- individuals
- companies with fewer than 100 employees
- internal testing and development purposes

The license text also states that the license converts to MIT on the defined change date.

## Packaged Linux build

Phant is also distributed as an official Linux packaged build.

That packaged build does not replace the license terms. For exact usage rights, rely on the repository license text.

## Source access and contribution

The source is available to inspect and build.

For contribution workflow details, use the main project repository and contribution guidance there.

## Important note

If you need an exact legal interpretation, use the license text itself rather than this summary.
