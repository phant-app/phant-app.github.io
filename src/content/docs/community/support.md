Use the support path that matches your need.

## Support channels

- Documentation
  Use the docs first for installation, onboarding, guides, and troubleshooting.

- GitHub Issues
  Use issues for bugs, broken behavior, and reproducible problems.
  https://github.com/phant-app/phant/issues

- GitHub Discussions
  Use discussions for general questions, ideas, and non-bug conversations.
  https://github.com/phant-app/phant/discussions

- Official website
  Use the official project website when you need the website-related support path referenced by the project.

## Bug report checklist

When opening an issue, include:

- your OS
- your Phant version
- steps to reproduce
- expected result
- actual result
- logs or screenshots if relevant
